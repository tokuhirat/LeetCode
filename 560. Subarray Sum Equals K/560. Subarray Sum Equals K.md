## STEP1

Time Limit Exceeded
```python
class Solution:
    def subarraySum(self, nums: List[int], k: int) -> int:
        cumulative_sum = [0]
        for num in nums:
            cumulative_sum.append(cumulative_sum[-1] + num)

        # subarray: (left, right] i.e. left: exclusive, right: inclusive
        count = 0
        for left in range(len(cumulative_sum) - 1):
            for right in range(left + 1, len(cumulative_sum)):
                subarray_sum = cumulative_sum[right] - cumulative_sum[left]
                if subarray_sum == k:
                    count += 1
        return count
```
n = len(nums) として  
時間計算量: $O(n^2)$  
空間計算量: $O(n)$  

- https://github.com/olsen-blue/Arai60/pull/16/files を見て通るコードは書けるようになった。
    - result という変数名は最適ではないが、count だと他の変数名と重複してわかりにくいと思った。frequency は大袈裟な印象がある。STEP2 で書き直す。
```python
from collections import defaultdict


class Solution:
    def subarraySum(self, nums: List[int], k: int) -> int:
        cumulative_sum = 0
        cumulative_sum_to_count = defaultdict(int)
        cumulative_sum_to_count[0] = 1
        result = 0
        for num in nums:
            cumulative_sum += num
            if cumulative_sum - k in cumulative_sum_to_count:
                result += cumulative_sum_to_count[cumulative_sum - k]
            cumulative_sum_to_count[cumulative_sum] += 1
        return result
```
時間計算量: $O(n)$  
空間計算量: $O(n)$
