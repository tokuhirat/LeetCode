# 112. Path Sum
## STEP1
- 何も見ずに解いてみる
- ただの二分木なので全ての path を確認する必要がありそう
- 合計を計算していき、leaf で target と等しいか確認する
- DFS を再帰で書くと、target - node.val となるパスが下流にあるかを確認する。stack で書くとこれまでの合計を記録していき、leaf で確認する。stack の方が直感的な感じがした。

```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def hasPathSum(self, root: Optional[TreeNode], targetSum: int) -> bool:
        if root is None:
            return False

        nodes_and_path_sum = [(root, root.val)]
        while nodes_and_path_sum:
            node, path_sum = nodes_and_path_sum.pop()
            if node.left is None and node.right is None:
                if path_sum == targetSum:
                    return True
                continue
            if node.left is not None:
                nodes_and_path_sum.append(
                    (node.left, path_sum + node.left.val)
                )
            if node.right is not None:
                nodes_and_path_sum.append(
                    (node.right, path_sum + node.right.val)
                )
        return False
```
#### memo
ノード数を n として  
時間計算量: $O(n)$  
空間計算量: $O(n)$

class Solution:
    def hasPathSum(self, root: Optional[TreeNode], targetSum: int) -> bool:
        if root is None:
            return False
        targetSum -= root.val
        if root.left is None and root.right is None:
            return targetSum == 0
        return (
            self.hasPathSum(root.left, targetSum) or
            self.hasPathSum(root.right, targetSum)
        )

```python
class Solution:
    def hasPathSum(self, root: Optional[TreeNode], targetSum: int) -> bool:
        if root is None:
            return False

        nodes_and_path_sum = [(root, 0)]
        while nodes_and_path_sum:
            node, path_sum = nodes_and_path_sum.pop()
            path_sum += node.val
            if node.left is None and node.right is None:
                if path_sum == targetSum:
                    return True
                continue
            if node.left is not None:
                nodes_and_path_sum.append((node.left, path_sum))
            if node.right is not None:
                nodes_and_path_sum.append((node.right, path_sum))
        return False
```
