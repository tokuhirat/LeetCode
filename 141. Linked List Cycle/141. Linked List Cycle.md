# 141. Linked List Cycle
## STEP1
- 何も見ずに解いてみる
```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, x):
#         self.val = x
#         self.next = None

class Solution:
    def hasCycle(self, head: Optional[ListNode]) -> bool:
        seen_node = set()
        while head is not None:
            if head in seen_node:
                return True
            seen_node.add(head)
            head = head.next
        return False
```

## STEP2
### プルリクやドキュメントを参照
- editorialにあったFloyd's Cycle Finding Algorithmを実装してみた
```python
class Solution:
    def hasCycle(self, head: Optional[ListNode]) -> bool:
        if head is None:
            return False
        
        slow = head
        fast = head.next
        while slow != fast:
            if fast is None or fast.next is None:
                return False
            slow = slow.next
            fast = fast.next.next
        return True
```
