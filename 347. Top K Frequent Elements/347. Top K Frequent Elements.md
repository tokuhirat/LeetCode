# 347. Top K Frequent Elements
## STEP1
- 何も見ずに解いてみる
```python
class Solution:
    def topKFrequent(self, nums: List[int], k: int) -> List[int]:
        num_to_count = {}
        for num in nums:
            if num in num_to_count:
                num_to_count[num] += 1
            else:
                num_to_count[num] = 1
        sorted_count = sorted(num_to_count.items(), key=lambda x: x[1], reverse=True)
        return [num for (num, count) in sorted_count[:k]]
```
#### memo
- numごとの個数をdictで集計して、valueで並び替える方法で実装した。
- defaultdictを使うと初めてみる値について場合分けが不要になるが、単純な処理なので使わないことにした。

n = len(nums)として  
時間計算量: $O(n\log n)$  
空間計算量: $O(n)$

```python
import heapq
from collections import defaultdict


class Solution:
    def topKFrequent(self, nums: List[int], k: int) -> List[int]:
        num_to_count = defaultdict(int)
        for num in nums:
            num_to_count[num] += 1
        return heapq.nlargest(k, num_to_count.keys(), key=num_to_count.get)
```
計算量 $O(n\log k)$
