# 322. Coin Change
## STEP1
- 何も見ずに解いてみる
- 金額が小さい順に、coinを1枚追加することで払える金額に遷移して払える額を大きくしていけば良い。逆に金額が大きい順に、coinを1枚減らした金額を払えるか再帰的に確かめる方法もある。
- 金額が小さい順の方がわかりやすいと思ったがどちらも書いてみる。
- 金額が小さい順に構築
```python
import math


class Solution:
    def coinChange(self, coins: List[int], amount: int) -> int:
        amount_to_num_coins = [math.inf] * (amount + 1)
        # math.inf if amount cannot be made up
        amount_to_num_coins[0] = 0
        for total in range(amount):
            for coin in coins:
                new_total = total + coin
                if new_total > amount:
                    continue
                amount_to_num_coins[new_total] = min(
                    amount_to_num_coins[new_total], amount_to_num_coins[total] + 1
                )
        if amount_to_num_coins[amount] == math.inf:
            return -1
        return amount_to_num_coins[amount]
```
- 金額が大きい順から分割
```python
import math
import functools


class Solution:
    def coinChange(self, coins: List[int], amount: int) -> int:
        @functools.cache
        def get_num_coins(amount: int) -> int:
            if amount == 0:
                return 0
            num_coins = math.inf
            for coin in coins:
                if amount < coin:
                    continue
                num_coins = min(num_coins, get_num_coins(amount - coin) + 1)
            return num_coins
        
        num_coins = get_num_coins(amount)
        if num_coins == math.inf:
            return -1
        return num_coins
```
