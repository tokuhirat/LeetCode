# 82. Remove Duplicates from Sorted List II
## STEP1
- 何も見ずに解いてみる
```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def deleteDuplicates(self, head: Optional[ListNode]) -> Optional[ListNode]:
        dummy = ListNode()
        last = dummy
        node = head

        while node is not None:
            if node.next is None:
                last.next = node
                return dummy.next

            val = node.val
            if val != node.next.val:
                last.next = node
                last = node
                node = node.next
                continue

            while node.val == val:
                node = node.next
                if node is None:
                    last.next = None
                    return dummy.next
        return dummy.next
```
#### memo
- 少し手間取った。番兵を置くことはわかったが、末尾に同じ値が続いている場合に、連結を切る（last.next = None）ことを忘れていた。
- 冗長な感じはしたが、シンプルにする方法はわからなかった。

時間計算量
- O(N)  
空間計算量
- O(1)

- 隣接で値が同じならnextを飛ばしていく書き方（continueしてelse節を書かない）  
    ```python
    class Solution:
        def deleteDuplicates(self, head: Optional[ListNode]) -> Optional[ListNode]:
            dummy = ListNode(next=head)
            tail = dummy
            node = head
            while node is not None and node.next is not None:
                if node.val != node.next.val:
                    tail = node
                    node = node.next
                    continue
                while node.next is not None and node.val == node.next.val:
                    node.next = node.next.next
                tail.next = node.next
                node = node.next
            return dummy.next
    ```

- 隣接で値が同じならnextを飛ばしていく書き方（if文をcontinueせずelse節を書く）  
    node = node.nextをまとめて書けるが、やや見にくい印象
