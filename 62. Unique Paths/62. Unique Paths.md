# 62. Unique Paths
## STEP1
- 何も見ずに解いてみる
- 数学的に解けるが、これは求められていなさそう。
```python
import math


class Solution:
    def uniquePaths(self, m: int, n: int) -> int:
        return math.factorial(m + n - 2) // math.factorial(m - 1) // math.factorial(n - 1)
```
- 再帰で書いてみる。
- 最初 m == 0 などとしており間違えた。
- m, n に対称性があるので大小関係を設定すると cache にあたりやすく計算量が小さくなるかもしれない。
```python
from functools import cache


class Solution:
    @cache
    def uniquePaths(self, m: int, n: int) -> int:
        if m == 1 and n == 1:
            return 1
        if m == 1:
            return self.uniquePaths(m, n - 1)
        if n == 1:
            return self.uniquePaths(m - 1, n)
        return self.uniquePaths(m - 1, n) + self.uniquePaths(m, n - 1)
```
- （STEP2 で他の人の解法を見たコメント）`if m == 1 or n == 1: return 1` でよい

#### memo
factorial 版
時間計算量: ~~O(m + n)~~ ←そんなに単純ではないらしい。  
空間計算量: $O(1)$  
再帰版  
時間計算量: $O(m * n)$  
空間計算量: $O(m * n)$

```python
class Solution:
    def uniquePaths(self, m: int, n: int) -> int:
        num_paths = [[0] * n for _ in range(m)]
        for row in range(m):
            num_paths[row][0] = 1
        for column in range(n):
            num_paths[0][column] = 1
        for row in range(1, m):
            for column in range(1, n):
                num_paths[row][column] = num_paths[row - 1][column] + num_paths[row][column - 1]
        return num_paths[-1][-1]
```
