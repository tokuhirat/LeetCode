# 349. Intersection of Two Arrays
## STEP1
- 何も見ずに解いてみる
- set を使うと簡単
```python
class Solution:
    def intersection(self, nums1: List[int], nums2: List[int]) -> List[int]:
        nums1 = set(nums1)
        nums2 = set(nums2)
        return list(nums1 & nums2)
```

- set を使わない場合は、それぞれソートして先頭から確認する
```python
class Solution:
    def intersection(self, nums1: List[int], nums2: List[int]) -> List[int]:
        nums1 = sorted(nums1)
        nums2 = sorted(nums2)
        
        intersections = []
        index1 = 0
        index2 = 0
        while index1 < len(nums1) and index2 < len(nums2):
            n1 = nums1[index1]
            n2 = nums2[index2]
            if n1 == n2:
                if not intersections or n1 != intersections[-1]:
                    intersections.append(n1)
                index1 += 1
                index2 += 1
                continue
            if n1 > n2:
                index2 += 1
            else:
                index1 += 1
        return intersections
```
最初 n1 と n2 を誤ってインクリメントしていたので、わかりにくい変数名は良くないことを実感。num1 にした方がわかりやすい。

#### memo
m = len(nums1), m = len(nums2)として  
set を使う場合
時間計算量: $O(n+m)$  
空間計算量: $O(n+m)$  
set を使わない場合  
時間計算量: $O(n\log n + m \log m)$  
空間計算量: $O(n+m)$

## STEP3
### 3回ミスなく書く
set は使わないパターンを練習する
```python
class Solution:
    def intersection(self, nums1: List[int], nums2: List[int]) -> List[int]:
        nums1 = sorted(nums1)
        nums2 = sorted(nums2)

        index1 = 0
        index2 = 0
        intersections = []
        while index1 < len(nums1) and index2 < len(nums2):
            num1 = nums1[index1]
            num2 = nums2[index2]
            if num1 < num2:
                index1 += 1
                continue
            if num1 > num2:
                index2 += 1
                continue
            common_num = num1
            intersections.append(common_num)
            while index1 < len(nums1) and nums1[index1] == common_num:
                index1 += 1
            while index2 < len(nums2) and nums2[index2] == common_num:
                index2 += 1
        return intersections
```
