# 1. Two Sum
## STEP1
- 何も見ずに解いてみる
```python
class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        sorted_nums_with_index = sorted([(num, i) for i, num in enumerate(nums)])
        
        def add(left, right):
            return sorted_nums_with_index[left][0] + sorted_nums_with_index[right][0]
        
        def get_original_indices(left, right):
            return [sorted_nums_with_index[left][1], sorted_nums_with_index[right][1]]

        left = 0
        right = len(sorted_nums_with_index) - 1
        while True:
            sum_ = add(left, right)
            if sum_ == target:
                return get_original_indices(left, right)
            if sum_ > target:
                right -=1
                continue
            left += 1
```
#### memo
- 見たことがある問題。ソートしてindexを両側から調べればよい。
- これまでだったら関数化しない処理を関数として書いてみた。関数化しない場合長い変数名で処理が追いにくい。関数化すると適切な関数名がつけられていたらわかりやすい。今回の場合は多少分かりやすくなったかもしれない。

n = len(nums) として  
時間計算量: $O(n\log n)$  
空間計算量: $O(n)$

```python
class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        sorted_nums_with_index = sorted([(num, i) for i, num in enumerate(nums)])
        
        def add(left, right):
            return sorted_nums_with_index[left][0] + sorted_nums_with_index[right][0]
        
        def get_original_indices(left, right):
            return [sorted_nums_with_index[left][1], sorted_nums_with_index[right][1]]

        left = 0
        right = len(sorted_nums_with_index) - 1
        while left < right:
            sum_ = add(left, right)
            if sum_ == target:
                return get_original_indices(left, right)
            if sum_ > target:
                right -=1
                continue
            left += 1
        raise ValueError(
            f"Cannot find two numbers such that they add up to target={target}"
        )
```

dict を使うバージョン  
こちらがシンプルかつわかりやすいと思った。
```python
class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        num_to_index = {}
        for i, num in enumerate(nums):
            complement = target - num
            if complement in num_to_index:
                return [i, num_to_index[complement]]
            num_to_index[num] = i
        raise ValueError(
            f"Cannot find two numbers such that they add up to target={target}"
        )
```
