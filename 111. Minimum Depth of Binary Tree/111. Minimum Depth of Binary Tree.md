# 111. Minimum Depth of Binary Tree
## STEP1
- 何も見ずに解いてみる
- Max depth の場合と同様に書ける。leaf を見つけ次第 return する
- DFS は相性が悪いと思う。BFS だと枝刈りのようになるが、DFS ではそうではない。
```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def minDepth(self, root: Optional[TreeNode]) -> int:
        if root is None:
            return 0
        
        nodes = [root]
        depth = 0
        while nodes:
            depth += 1
            next_nodes = []
            for node in nodes:
                if node.left is None and node.right is None:
                    return depth
                if node.left is not None:
                    next_nodes.append(node.left)
                if node.right is not None:
                    next_nodes.append(node.right)
            nodes = next_nodes
```
#### memo
時間計算量: $O(n)$  
空間計算量: $O(n)$

DFS (再帰): 帰りがけ
```python
class Solution:
    def minDepth(self, root: Optional[TreeNode]) -> int:
        if root is None:
            return 0
        if root.left is None and root.right is None:
            return 1
        if root.left is None:
            return self.minDepth(root.right) + 1
        if root.right is None:
            return self.minDepth(root.left) + 1
        return min(self.minDepth(root.left), self.minDepth(root.right)) + 1
```
DFS (stack): 行きがけ
```python
import sys


class Solution:
    def minDepth(self, root: Optional[TreeNode]) -> int:
        if root is None:
            return 0
        
        min_depth = sys.maxsize
        nodes = [(root, 1)]
        while nodes:
            node, depth = nodes.pop()
            if depth >= min_depth:
                continue
            if node.left is None and node.right is None:
                min_depth = depth
                continue
            for child in [node.left, node.right]:
                if child is not None:
                    nodes.append((child, depth + 1))
        return min_depth
```

DFS (再帰): 行きがけ。行きがけだと枝刈りできる。わかりやすくはない。
```python
class Solution:
    def minDepth(self, root: Optional[TreeNode]) -> int:
        if root is None:
            return 0

        min_depth = sys.maxsize

        def minDepthHelper(node: Optional[TreeNode], depth: int):
            nonlocal min_depth
            if node is None:
                return
            if node.left is None and node.right is None:
                min_depth = min(min_depth, depth)
                return
            if depth >= min_depth:
                return
            minDepthHelper(node.left, depth + 1)
            minDepthHelper(node.right, depth + 1)

        minDepthHelper(root, 1)
        return min_depth
```

DFS (stack): 帰りがけ。とてもわかりにくい。  
down を取り出したら行きがけであり、leaf でない場合、深さを +1 して 子を stack に積む。   
leaf にたどり着いたら depth が確定し、up と印をつけて stack に積む。  
up を取り出したら帰りがけであり、親が同じ node がいて、down ならば探索するために stack 中の位置を入れ替え、up ならば depth の小さい方で stack に積まれている 親を up に変更。
親が同じ node がいないならば、depth を伝えて stack に積まれている 親を up に変更。  


コードを書きながら left と right をタプルにして stack に積もうとしたりしてだいぶ迷走した。  
手で作業して言語化したら書けた。
真面目に書くなら namedtuple で stack に積むと思います。
https://docs.python.org/3/library/collections.html#collections.namedtuple

```python
class Solution:
    def minDepth(self, root: Optional[TreeNode]) -> int:
        if root is None:
            return 0
        
        def update_depth_post_order(depth):
            stack_top = stack.pop()
            stack_top[0] = "up"
            stack_top[3] = depth
            stack.append((stack_top))
        
        stack = [["down", root, None, 1]]  # [direction, node, parent, depth]
        while stack:
            direction, node, parent, depth = stack.pop()
            if direction == "down":                
                if node.left is None and node.right is None:
                    stack.append(["up", node, parent, depth])
                    continue

                stack.append([direction, node, parent, depth])
                if node.left is not None:
                    stack.append(["down", node.left, node, depth + 1])
                if node.right is not None:
                    stack.append(["down", node.right, node, depth + 1])
            else:
                if parent is None:
                    return depth

                if stack[-1][2] is parent and stack[-1][0] == "down":
                    # another children is not traversed
                    # swap to traverse another children
                    another_children = stack.pop()
                    stack.append([direction, node, parent, depth])
                    stack.append(another_children)
                    continue
                
                if stack[-1][2] is parent and stack[-1][0] == "up":
                    # two children traversed
                    _, _, _, another_depth = stack.pop()
                    min_child_depth = min(depth, another_depth)
                    update_depth_post_order(min_child_depth)
                    continue
                
                # one child exists
                update_depth_post_order(depth)
```
