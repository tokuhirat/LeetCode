# 703. Kth Largest Element in a Stream
## STEP1
- 何も見ずに解いてみる
- heapqが使えそうだと思い、[ドキュメント](https://docs.python.org/3.13/library/heapq.html)を見ながら書いたが、計算量がわかっていないのでTime Limit Exceededになった。
```python
import heapq


class KthLargest:
    def __init__(self, k: int, nums: List[int]):
        self.heapq = nums
        heapq.heapify(self.heapq)
        self.k = k

    def add(self, val: int) -> int:
        heapq.heappush(self.heapq, val)
        return heapq.nlargest(self.k, self.heapq)[-1]
```

答えを見て書き直した。
```python
class KthLargest:
    def __init__(self, k: int, nums: List[int]):
        self.min_heap = []
        self.k = k

        for num in nums:
            self.add(num)

    def add(self, val: int) -> int:
        if len(self.min_heap) < self.k or self.min_heap[0] < val:
            heapq.heappush(self.min_heap, val)
            if len(self.min_heap) > self.k:
                heapq.heappop(self.min_heap)
        return self.min_heap[0]
```

#### memo

M: len(nums), N: addが呼ばれる回数として  
時間計算量: $O((M+N)\cdot\log k)$  
&emsp;heapqはバイナリヒープで、追加と削除は $O(\log k)$でできる。  
空間計算量: $O(k)$

- 毎回ソートしているのもおかしそう。ほぼソートされているlistのソートは挿入ソートであれば速い？Timsort, powersortは調べきれていない。
```python
class KthLargest:
    def __init__(self, k: int, nums: List[int]):
        self.k = k
        self.sorted_nums = nums[:]
        self._sort_top_k_nums()  # len(self.sorted_nums) == k or k - 1
    
    def _sort_top_k_nums(self):
        self.sorted_nums.sort(reverse=True)
        self._remove_num_under_threshold()

    def _remove_num_under_threshold(self):
        while len(self.sorted_nums) > self.k:
            self.sorted_nums.pop()

    def add(self, val: int) -> int:
        if len(self.sorted_nums) < self.k or val > self.sorted_nums[-1]:
            self.sorted_nums.append(val)
            self._sort_top_k_nums()
        return self.sorted_nums[-1]
```

k の制約がない場合、初期入力のnumsがk個に満たない場合に足りない分を -inf みたいな値で埋める方法もあるかもしれないと思った。  
kの制約なしバージョンを書いてみた。計算量 $O(M\log M+Nk)$
```python
LOWEST_SCORE = -100_000

class KthLargest:
    def __init__(self, k: int, nums: List[int]):
        self.k = k
        self.sorted_nums = nums[:]
        self.sorted_nums.sort(reverse=True)
        if len(self.sorted_nums) < k:
            empty_nums = [LOWEST_SCORE] * (k - len(nums))
            self.sorted_nums += empty_nums
        else:
            self.sorted_nums = self.sorted_nums[:k]

    def add(self, val: int) -> int:
        if val > self.sorted_nums[-1]:
            for i, num in enumerate(self.sorted_nums):
                if val > num:
                    self.sorted_nums.insert(i, val)
                    break
            self.sorted_nums.pop()
        return self.sorted_nums[-1]
```

テスト例。
```python
obj = KthLargest(3, [50])
for num in [20, 30]:
    print("new score", num)
    kth_val = obj.add(num)
    print("top k vals", obj.sorted_nums)
    print("kth val", kth_val)
    print()
```

## STEP3
### 3回ミスなく書く
```python
import heapq

class KthLargest:
    def __init__(self, k: int, nums: List[int]):
        self.top_k_values = []  # min heap
        self.k = k

        for num in nums:
            self.add(num)

    def add(self, val: int) -> int:
        if len(self.top_k_values) < self.k or val > self.top_k_values[0]:
            heapq.heappush(self.top_k_values, val)
            if len(self.top_k_values) > self.k:
                heapq.heappop(self.top_k_values)
        return self.top_k_values[0]
```
