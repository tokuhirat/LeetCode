# 103. Binary Tree Zigzag Level Order Traversal
## STEP1
- 何も見ずに解いてみる
- 102\. Binary Tree Level Order Traversal とほぼ同じ。level が変わるごとに出力する順番を逆転させる。BFS でないと難しそう。ジグザグなので nodes を毎回反転させればよい（正しくない）。
- もしくは、問題102 と同じ方法で解き、偶数番目の level をひっくり返せばよい。
- 最初、nodes = next_nodes[::-1] としていて間違った。left と right のアクセス順が変わっていないので正しくない。
- 修正して level の偶奇で反転させるようにした。

```python
class Solution:
    def zigzagLevelOrder(self, root: Optional[TreeNode]) -> List[List[int]]:
        if root is None:
            return []
        
        nodes = [root]
        level_ordered_values = []
        level = 0
        while nodes:
            values = []
            next_nodes = []
            for node in nodes:
                values.append(node.val)
                if node.left is not None:
                    next_nodes.append(node.left)
                if node.right is not None:
                    next_nodes.append(node.right)
            if level % 2 == 1:
                values = values[::-1]
            level_ordered_values.append(values)
            nodes = next_nodes
            level += 1
        return level_ordered_values
```

```python
class Solution:
    def zigzagLevelOrder(self, root: Optional[TreeNode]) -> List[List[int]]:
        if root is None:
            return []
        
        level_ordered_values = []
        nodes = [root]
        while nodes:
            values = []
            next_nodes = []
            for node in nodes:
                values.append(node.val)
                if node.left is not None:
                    next_nodes.append(node.left)
                if node.right is not None:
                    next_nodes.append(node.right)
            level_ordered_values.append(values)
            nodes = next_nodes
        
        zigzag_result = []
        for level, values in enumerate(level_ordered_values):
            if level % 2 == 0:
                zigzag_result.append(values)
            else:
                zigzag_result.append(values[::-1])
        return zigzag_result
```
