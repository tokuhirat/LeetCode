## STEP1
- 何も見ずに解いてみる
- はじめ色についてもループ必要かと思ったが、対称性があるので不要と気がついた。
- 変数名がわかりやすいとは思わないが他のいい案もないので、Step2で修正する。
```python
class Solution:
    def numWays(self, n: int, k: int) -> int:
        if n <= 0:
            raise ValueError("n must be positive")
        if n <= 2:
            return k ** n
        last_two_same_color = k
        last_two_different_color = k * (k - 1)
        for _ in range(n - 2):
            next_last_two_same_color = last_two_different_color
            total = last_two_same_color + last_two_different_color
            next_last_two_different_color = total * (k - 1)
            last_two_same_color = next_last_two_same_color
            last_two_different_color = next_last_two_different_color
        return last_two_same_color + last_two_different_color
```
#### memo
時間計算量: $O(n)$  
空間計算量: $O(1)$

書き直し
```python
class Solution:
    def numWays(self, n: int, k: int) -> int:
        if n <= 0:
            raise ValueError("n must be positive")
        if k <= 0:
            raise ValueError("k must be positive")
        
        if n <= 2:
            return k ** n
        
        one_consecutive = k * (k - 1)  # ways to paint last two fence with different colors
        two_consecutive = k  # ways to paint last two fence with same color
        for _ in range(3, n + 1):
            temp_one = one_consecutive
            temp_two = two_consecutive
            one_consecutive = (temp_one + temp_two) * (k - 1)
            two_consecutive = temp_one
        return one_consecutive + two_consecutive
```
