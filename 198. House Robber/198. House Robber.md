# 198. House Robber
## STEP1
- 何も見ずに解いてみる
- 奪った場合と奪ってない場合の稼ぎをそれぞれ管理するとよい。
```python
class Solution:
    def rob(self, nums: List[int]) -> int:
        max_gain_robbed = nums[0]
        max_gain_not_robbed = 0
        for i in range(1, len(nums)):
            prev_gain_robbed = max_gain_robbed
            max_gain_robbed = max_gain_not_robbed + nums[i]
            max_gain_not_robbed = max(prev_gain_robbed, max_gain_not_robbed)
        return max(max_gain_robbed, max_gain_not_robbed)
```
#### memo
時間計算量: $O(n)$  
空間計算量: $O(1)$

再帰のパターン
```python
import functools


class Solution:
    def rob(self, nums: List[int]) -> int:
        @functools.cache
        def max_gain_from_index(i: int) -> int:
            if i == 0:
                return nums[0]
            if i == 1:
                return max(nums[:2])
            return max(max_gain_from_index(i - 1), max_gain_from_index(i - 2) + nums[i])
        
        return max_gain_from_index(len(nums) - 1)
```

配列を使うパターン
```python
class Solution:
    def rob(self, nums: List[int]) -> int:
        if len(nums) <= 2:
            return max(nums, default=0)
        max_gain = [0] * len(nums)
        max_gain[0] = nums[0]
        max_gain[1] = max(nums[:2])
        for i in range(2, len(nums)):
            max_gain[i] = max(max_gain[i - 1], max_gain[i - 2] + nums[i])
        return max_gain[-1]
```

## STEP3
### 3回ミスなく書く
```python
class Solution:
    def rob(self, nums: List[int]) -> int:
        max_gain_robbed = nums[0]
        max_gain_skipped = 0
        for i in range(1, len(nums)):
            prev_max_gain_robbed = max_gain_robbed
            max_gain_robbed = max_gain_skipped + nums[i]
            max_gain_skipped = max(max_gain_skipped, prev_max_gain_robbed)
        return max(max_gain_skipped, max_gain_robbed)
```
