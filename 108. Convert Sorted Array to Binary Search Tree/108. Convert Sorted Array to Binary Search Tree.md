# 108. Convert Sorted Array to Binary Search Tree
## STEP1
- 何も見ずに解いてみる
- 二分探索木はあまりなれていない。動的にやるのは難しそう。与えられた列から構築するだけなら中央の値を取ってくるを繰り返せば良さそう。
- 入力が空の場合はNoneを返せば良さそう。
- 再帰に慣れてきた感じがある。
```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def sortedArrayToBST(self, nums: List[int]) -> Optional[TreeNode]:
        if not nums:
            return None
        
        middle = len(nums) // 2
        root = TreeNode(nums[middle])
        root.left = self.sortedArrayToBST(nums[:middle])
        root.right = self.sortedArrayToBST(nums[middle + 1:])
        return root
```
#### memo
n = len(nums) として  
時間計算量: $O(n \log n)$    
空間計算量: $O(n)$

```python
class Solution:
    def sortedArrayToBST(self, nums: List[int]) -> Optional[TreeNode]:        
        def construct_BST_from_index(lo, hi) -> Optional[TreeNode]:
            if lo >= hi:
                return None
            middle = (lo + hi) // 2
            root = TreeNode(nums[middle])
            root.left = construct_BST_from_index(lo, middle)
            root.right = construct_BST_from_index(middle + 1, hi)
            return root

        return construct_BST_from_index(0, len(nums))
```

空間計算量: $O(\log n)$  

stack を使う方法
```python
class Solution:
    def sortedArrayToBST(self, nums: List[int]) -> Optional[TreeNode]:
        if not nums:
            return None
        
        dummy = TreeNode()
        stack = [(dummy, "left", (0, len(nums)))]
        while stack:
            parent, direction, (lo, hi) = stack.pop()
            if lo >= hi:
                continue
            middle = (lo + hi) // 2
            node = TreeNode(nums[middle])
            setattr(parent, direction, node)
            stack.append((node, "left", (lo, middle)))
            stack.append((node, "right", (middle + 1, hi)))
        return dummy.left
```
