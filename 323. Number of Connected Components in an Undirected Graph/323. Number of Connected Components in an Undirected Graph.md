# 323. Number of Connected Components in an Undirected Graph
## STEP1
- 何も見ずに解いてみる
- Number of Islands とほぼ同じだが、遷移先（エッジ）が与えられている。隣接リストに変形すれば扱いやすい。
- BFS, DFS, UnionFind どれでも良さそう。記述量が少なく、書きやすいBFSで書く。
- adjacents_lists は標準的な名前か自信がない。STEP2で適宜修正する。

```python
from collections import deque


class Solution:
    def countComponents(self, n: int, edges: List[List[int]]) -> int:
        adjacents_lists = [[] for _ in range(n)]
        for (node_i, node_j) in edges:
            adjacents_lists[node_i].append(node_j)
            adjacents_lists[node_j].append(node_i)
        
        def traverse_connected_component(node: int):
            node_to_visit = deque([node])
            visited.add(node)
            while node_to_visit:
                node = node_to_visit.popleft()
                for adjacent_node in adjacents_lists[node]:
                    if adjacent_node in visited:
                        continue
                    node_to_visit.append(adjacent_node)
                    visited.add(adjacent_node)

        visited = set()
        num_connected_components = 0
        for node in range(n):
            if node in visited:
                continue
            num_connected_components += 1
            traverse_connected_component(node)
        return num_connected_components
```
#### memo
E = len(deges) として  
時間計算量: $O(E+n)$  
空間計算量: $O(E+n)$
