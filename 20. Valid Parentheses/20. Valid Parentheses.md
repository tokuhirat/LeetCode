# 20. Valid Parentheses
## STEP1
- 何も見ずに解いてみる
```python
class Solution:
    def isValid(self, s: str) -> bool:
        close_to_open_braket = {")": "(", "}": "{", "]": "["}
        stack = []
        for si in s:
            if si in close_to_open_braket.values():
                stack.append(si)
                continue
            supposed_braket = close_to_open_braket[si]
            if not stack:
                return False
            actual_braket = stack.pop()
            if supposed_braket != actual_braket:
                return False
        return not stack

```
#### memo
- stackが使えることを知っていた。かっこごとにopenとcloseの対応をmapするとコードはシンプルだが、keyがcloseでvalueがopenなのは直感的ではないと思った。

時間計算量
- O(N)  
空間計算量
- O(N)

## STEP3
### 3回ミスなく書く
かっこ以外の入力がある場合も許容し、かっこの対応付をする方針にした
```python
class Solution:
    def isValid(self, s: str) -> bool:
        open_to_close_braket = {"(": ")", "{": "}", "[": "]"}
        stack = []
        for si in s:
            if si in open_to_close_braket:
                stack.append(si)
                continue
            if si not in open_to_close_braket.values():
                continue
            if not stack:
                return False
            actual_open_braket = stack.pop()
            supposed_close_braket = open_to_close_braket[actual_open_braket]
            if supposed_close_braket != si:
                return False
        return not stack

```
