# 104. Maximum Depth of Binary Tree
## STEP1
- 何も見ずに解いてみる
- DFS, BFS どちらでも解けそうなので両方書いてみる。
- root が None のときどうしたら良いかわからないが0とする
- 再帰は recursionlimit に注意しないといけないので積極的には使いたくない。深さなのでBFSが自然だと思った。
- BFSで書く場合、グラフでなく木構造だとループを階層ごとに回す方がわかりやすい気がする。（queue にdepthを追加しない書き方）  

BFSバージョン
```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def maxDepth(self, root: Optional[TreeNode]) -> int:
        if root is None:
            return 0

        nodes = [root]
        depth = 0
        while nodes:
            depth += 1
            next_nodes = []
            for node in nodes:
                if node.left is not None:
                    next_nodes.append(node.left)
                if node.right is not None:
                    next_nodes.append(node.right)
            nodes = next_nodes
        return depth
```
DFS（再帰）バージョン
```python
class Solution:
    def maxDepth(self, root: Optional[TreeNode]) -> int:
        def depth_of_subtree(node: Optional[TreeNode]) -> int:
            if node is None:
                return 0
            left_subtree_depth = depth_of_subtree(node.left)
            right_subtree_depth = depth_of_subtree(node.right)
            return max(left_subtree_depth, right_subtree_depth) + 1
        
        return depth_of_subtree(root)
```
DFS（stack）バージョン
```python
class Solution:
    def maxDepth(self, root: Optional[TreeNode]) -> int:
        if root is None:
            return 0

        max_depth = 0
        stack = [(root, 1)]
        while stack:
            node, depth = stack.pop()
            max_depth = max(max_depth, depth)
            if node.left is not None:
                stack.append((node.left, depth + 1))
            if node.right is not None:
                stack.append((node.right, depth + 1))
        return max_depth
```

#### memo
時間計算量: $O(n)$  
空間計算量: $O(n)$  

## STEP2
### プルリクやドキュメントを参照
- DFS (再帰)は maxDepth 自体を使って書ける。  
rootという引数名が微妙でどのnodeを入れるかは呼び出し側によるのでは。Tree のクラスとして実装するならget_tree_height(self) のようなメソッドを作ってその中で self.maxDepth(self.root) を呼ぶようにすると思う。
```python
class Solution:
    def maxDepth(self, root: Optional[TreeNode]) -> int:
        if root is None:
            return 0
        left_depth = self.maxDepth(root.left)
        right_depth = self.maxDepth(root.right)
        return max(left_depth, right_depth) + 1
```
